# 📊 Sales Dashboard (Power BI)

## 📌 Overview
This project is an interactive Power BI dashboard designed to analyze sales performance and generate business insights through data visualization.

---

## 🛠 Tools Used
- Power BI

---

## 📊 Dashboard Features
- Interactive filters (Region, Category, Subcategory)
- KPI metrics (Total Sales, Profit, Quantity)
- Visual charts for trend and performance analysis
- Clean and user-friendly layout

---

## 🔍 Key Insights
- Identified top-performing regions based on sales
- Analyzed category-wise performance
- Highlighted key sales trends
- Compared profit across different segments
